# ☀️ SOLARIS — Solana Token Intelligence

Analisi token Solana in tempo reale. Risk scoring, liquidità, trending detection.

## 🚀 Come vederlo

### Opzione 1: GitHub Pages (online)

Il sito è già deployato su GitHub Pages:

**🔗 https://lolarok.github.io/solaris/**

### Opzione 2: Locale (il tuo computer)

1. **Clona il repository:**
   ```bash
   git clone https://github.com/Lolarok/solaris.git
   cd solaris
   ```

2. **Apri nel browser:**
   - **Mac:** `open index.html`
   - **Windows:** `start index.html`
   - **Linux:** `xdg-open index.html`

   Oppure trascina `index.html` nel browser.

3. **Con un server locale (opzionale):**
   ```bash
   # Python
   python3 -m http.server 8080

   # Node.js (se hai npx)
   npx serve .

   # Poi apri http://localhost:8080
   ```

### Opzione 3: VS Code Live Server

1. Apri la cartella in VS Code
2. Installa l'estensione "Live Server"
3. Clicca "Go Live" in basso a destra

## ✨ Funzionalità

- Risk scoring per token Solana
- Metriche di liquidità
- Trending token detection
- Aggiornamento in tempo reale
- Dashboard dark mode

- HTML + CSS + JavaScript vanilla
